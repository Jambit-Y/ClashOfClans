## 🎯 核心功能与规划 (Features & Planning)

### I. 基础功能 (Base Implementation)

**本阶段必须完成，以确保游戏具备核心可玩性。**

#### 1. 主村庄与资源系统
- 主村庄/大本营系统
- 三类资源：金币、圣水、人口/容量
- 支持常见建筑类型：资源生成建筑、军营、防御塔、存储建筑

#### 2. AI 战斗系统
- 路径寻找：兵种自动寻路
- 攻击判定：兵种自动战斗与攻击距离判定
- 建筑优先级：实现兵种的攻击优先级
- 建筑等级：支持至少 3 级的建筑升级

#### 3. 战斗流程与兵种
- 一局战斗流程：投放兵种、自动战斗、胜负判定
- 支持常见兵种：弓箭手、野蛮人、炸弹人、巨人
- 地图支持：至少 2 张不同地图
- 音效支持：背景音乐与战斗音效

### II. 可选功能 (Optional Features)

- 多人进攻系统
- 联盟系统（可加入部落）
- 部落战 / 联机对战
- 天空兵种（飞龙等）
- 陷阱、炸弹系统
- 回放系统
- 建筑施工时间（加速水晶机制）

## 🚀 本地环境设置与构建 (Local Setup & Build)

1. **克隆项目**
   ```sh
   git clone https://github.com/JambitX/ClashOfClans.git
   cd ClashOfClans
   ```

2. **安装依赖**
   - 安装 [Cocos2d-x](https://www.cocos.com/en/cocos2dx) 并配置环境变量
   - 安装 [CMake](https://cmake.org/download/)
   - 安装 Visual Studio 2022（含 C++ 开发工具集）

3. **生成工程文件**
  
   ``` cmd
   cmake -S . -B build -G "Visual Studio 17 2022" -A win32 -T v143
   ```

4. **编译与运行**
   - 用 Visual Studio 打开 `build` 目录下的解决方案文件（.sln）
   - 编译并运行主程序
